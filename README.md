# Wedding Landing Page (Static Website)

A ready-to-edit static website for a wedding landing page, built with:
- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` → Main landing page
- `css/style.css` → Styles and responsive layout
- `js/main.js` → Mobile menu and countdown logic
- `data/wedding-info.json` → Editable wedding content reference
- `assets/images/` → Photos and backgrounds
- `assets/icons/` → Icons and logos
- `assets/fonts/` → Local font files (optional)

## Quick Start

1. Open `index.html` in your browser.
2. Edit couple names, dates, and content directly in `index.html`.
3. Update styles in `css/style.css`.
4. Update countdown date from `window.WEDDING_DATE` in `index.html`.
